Log 4 (Internet Architecture)

On the 4th lesson of our NVQ we covered Client servers, protocols, the backbone of the internet, shared domains , cloud computing and RSS. Firstly we learned about client servers which uses a central server where many computers connect the internet or a network. The general hardware and software items we will need is a PC, a network card, a modem, a router and the initial internet browser. There are also several bandwidth technologies that can be used which are PSTN (public switch telephone network) , ISDN (integrated services digital network), and DSL (digital subscriber line). Next we learnt about several protocols which state how data is sent over the internet. Transmission Control Protocol (TCP) defines how packets are created ready to be sent via IP and then put back together by the recipient. User Datagram Protocol (UDP) are essentially smaller packets without error checking. Simple Mail Transfer Protocol (SMTP) is creating and sending email packets. Post Office Protocol Version 3 (POP3) is the receiving and checking end of email packets. Hypertext Transfer Protocol (HTTP) uses TCP/IP to transfer HTML and File Transfer Protocol (FTP) uses TCP/IP to transfer files. Next we learnt about the OSI model which is the Open Systems Interconnection which is a model that defines the networking framework to implement protocols in seven different layers. These layers are (bottom) Physical, Data Link, Network, Transport, Session, Presentation and Application. After this we learnt about the Internet Backbone which entails point presencing and Point connections. Point-to-Point (PPP) is a data link protocol used to establish a direct connection between two nodes. It can provide connection authentication , transmission and encryption. The internet backbone can therefore be described as principle data routes between large strategically interconnected computer networks and core router on the internet. These data routes are normally hosted by commercial, government and high capacity network centres.
